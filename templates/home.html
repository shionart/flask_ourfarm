Getting started.

  1. Hubungkan perangkat ini dengan WiFi
  2. Kunjungi bwcr.rizaldiariif.com untuk melakukan Registrasi
  3. Tambah Nodes di menu Arduino - Tambah Perangkat dengan keterangan sebagai berikut :
    • IP Address : {{ip}}
    • Device Type : Garden
    • SSID WiFi : "nama WiFi anda yang tersambung dengan perangkat ini"
    • Password WiFi : "password WiFi anda tersambung dengan perangkat ini"
  4. Download dan install file.ino yang disediakan setelah menambah perangkat pada arduino yang akan digunakan!
  5. Hubungkan arduino dengan aliran listrik, tunggu sampai alat mengirim data (cek di sini).
