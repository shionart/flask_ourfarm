Panduan Memulai

  1. Hubungkan perangkat ini dengan WiFi
  2. Kunjungi bwcr.insightdata.xyz untuk melakukan Registrasi
  3. Tambah Perangkat jenis "Taman" di menu Arduino - Tambah Perangkat dengan keterangan sebagai berikut :
    • Alamat IP : {{ip}}
    • Device Type : Garden
    • SSID WiFi : "nama WiFi anda yang tersambung dengan perangkat ini"
    • Password WiFi : "password WiFi anda tersambung dengan perangkat ini"
  4. Download dan install file.ino yang disediakan setelah menambah perangkat pada arduino yang akan digunakan!
  5. Salin User ID pada bagian card Setup di bawah fitur download file.ino dan tempelkan di form bawah ini
  6. Hubungkan arduino dengan aliran listrik, tunggu sampai alat mengirim data (cek di sini).
